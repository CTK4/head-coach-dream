# Schedule / Slate UI

## Routes
- `/hub/schedule` (tabbed schedule landing)
- `/hub/schedule/week/:weekNumber` (week slate)
- `/hub/schedule/team/:teamId` (team schedule)
- `/hub/schedule/game/:gameKey` (game details)

## Data Sources
- `state.hub.schedule` for matchups.
- `state.league.results` for finals.
- `state.gameHistory` for user-played box score drilldown.
